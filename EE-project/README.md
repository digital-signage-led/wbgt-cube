# WBGT サイネージ — 仙台市・夢メッセみやぎ（EE-project）

## どの HTML を使うか

| ファイル | 用途 | 開き方 | 備考 |
|----------|------|--------|------|
| **`index.html`** | **本番用**（メイン本体） | `index.html` をブラウザで開く | 環境省 WBGT・Open-Meteo から**実データ**を取得 |
| **`demo.html`** | **デモ用**（入口のみ） | `demo.html` を開く | 中身はなく、自動で `index.html?demo=1` へ転送 |
| **`index.html?demo=1`** | **デモ表示**（本体は index.html） | `demo.html` 経由、または URL に `?demo=1` | サンプルデータ。警戒レベル5段階を順に巡回 |

**表示の実体は常に `index.html` です。** `demo.html` はデモモードへ誘導するショートカットだけです。
